# Домашнее задание

Научить компилятор вычислять дисперсию (variance) отфильтрованных данных по формуле

Var[x] = M[x²] − (M[x])²

M[x²] = (Σxᵢ²)/N — среднее квадрата

M[x] = (Σxᵢ)/N — среднее
